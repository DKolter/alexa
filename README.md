# Alexa
A home automation program to control infrared lights and open the door using voice commands in german. Name was shamelessly stolen from
amazon, because it is best recognized by the vosk model running local on my raspberry pi, which doesn't have enough processing power for 
a larger language model that could recognize other names as efficient.

For the speaker I bought a conference microphone from amazon, which provides a decent range (3m). It works from 
any room in our apartment (even though a little shouting is needed sometimes).
